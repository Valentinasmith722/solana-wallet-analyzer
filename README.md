# 🔍 Solana Wallet Analyzer

> **Track your entire Solana portfolio in one dashboard.**
> 
> PnL, token performance, NFT holdings, DeFi positions, airdrop eligibility.
> 
> **100% free. No API keys. Open source.**

---

## Features

| Feature | Description |
|---------|-------------|
| 📊 **Portfolio Overview** | Total value, 24h change, asset allocation |
| 💰 **PnL Tracking** | Realized & unrealized gains per token |
| 🎨 **NFT Gallery** | Floor prices, rarity scores, collection analytics |
| 🏦 **DeFi Positions** | Lending, borrowing, LP positions across protocols |
| 🎁 **Airdrop Eligibility** | Check if you qualify for upcoming airdrops |
| 📈 **Performance Charts** | Historical portfolio value, benchmark vs SOL |
| 🔍 **Transaction History** | Categorized transactions with cost basis |
| 🚨 **Risk Alerts** | Liquidation warnings, impermanent loss alerts |

## Supported Protocols

- Jupiter (swaps, limit orders)
- Marinade (staking)
- Kamino (lending, yield)
- Jito (liquid staking)
- Raydium (LP, farms)
- Orca (concentrated liquidity)
- Drift (perpetuals)
- Solend (lending)
- Meteora (dynamic AMM)
- Tensor (NFT trading)

## Quick Start

```bash
git clone https://github.com/Valentinasmith722/solana-wallet-analyzer.git
cd solana-wallet-analyzer
bun install
bun run dev
```

Open http://localhost:3000 and enter any Solana wallet address.

## Privacy

This tool runs entirely in your browser. No data is sent to any server. Your wallet address is only used to query public blockchain data via RPC.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Solana Web3.js
- Chart.js for visualizations
